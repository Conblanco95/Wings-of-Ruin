# ⚔ Wings of Ruin — An Empyrean Choose Your Own Adventure

A fully AI-narrated interactive fiction game set in the world of Rebecca Yarros' Empyrean series (Fourth Wing, Iron Flame, Onyx Storm). Every scene is dynamically generated by Claude, with mood-reactive visuals, particle effects, and branching storylines that lead to 7 possible endings.

---

## Prerequisites

- **Node.js 18+** — [download here](https://nodejs.org/)
- **An Anthropic API key** — [get one here](https://console.anthropic.com/settings/keys)

---

## Setup (5 minutes)

### 1. Install dependencies

```bash
cd wings-of-ruin
npm install
```

### 2. Add your API key

Create a file called `.env` in the project root:

```
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

Or copy the template:

```bash
cp .env.example .env
```

Then edit `.env` and paste your real key.

### 3. Run the game

```bash
npm run dev
```

This starts two things:
- **API proxy** on `http://localhost:3001` (handles your API key securely)
- **Game UI** on `http://localhost:5173` (open this in your browser)

Open **http://localhost:5173** and play!

---

## How It Works

```
Browser (React)  →  Local proxy (Express)  →  Anthropic API (Claude Sonnet)
    port 5173            port 3001               api.anthropic.com
```

Your API key never touches the browser — it stays on the local Express server. The React frontend sends requests to `/api/messages`, which the proxy forwards to Anthropic with your key attached.

## How to Play

- The game generates a random Sorrengail rider name for you each playthrough
- Read the scene, then click one of the choices (A, B, or C)
- Your decisions are tracked — they affect the story and determine your ending
- The story naturally reaches a climax after 12–20 choices
- Click **↺ Restart** at any time for a fresh playthrough with a new name

### Endings

| Ending | How to get it |

| 🏔️ The Exile's Peace | Choose Aretia and freedom over Navarre |
| ⚖️ The Cost of Comfort | Stay within the system, make painful compromises |
| 🌊 The Impossible Alliance | Broker peace between Navarre, Poromiel, and the Archipelago |
| 🌩️ After the Storm | Survive the convergence and shatter the venin command |

## Cost

Each scene costs roughly **$0.01–0.03** in API usage (Claude Sonnet). A full playthrough of ~15 choices costs approximately **$0.15–0.45**.

You can switch to a cheaper model by editing `src/App.jsx` and changing the model string:
- `claude-sonnet-4-20250514` — best quality (default)
- `claude-haiku-4-5-20251001` — faster and cheaper, slightly less rich prose

## Troubleshooting

| Problem | Fix |
|---------|-----|
| "Could not reach the narrator" | Make sure the server is running (`npm run dev`) |
| "Invalid API key" | Check your `.env` file has a valid `ANTHROPIC_API_KEY` |
| Scene text looks garbled | Hit "Try Again" — occasionally the model's JSON output needs a retry |
| Blank screen | Open browser console (F12) for error details |
